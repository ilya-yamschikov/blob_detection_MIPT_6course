В папке два теста. 
1. 'sunflower.png' - реальное изображение для поиска темных пятен.
2. 'generate_sample.m' - функция генерирующая изображение с набором светлых пятен заданной формы.
Аргументы: 
	a) mesh_size - размер изображения в пикселах (генерируется изображение mesh_size x mesh_size)
	b) blobs_count - количество светлых пятен на изображении.
	c) type - тип пятен. В текущей версии поддерживаются значения:
		- 'blob' - двумерная гауссиана
		- 'circle' - окружность (с интенсивностью 1)
	d) apply_noise - добавлять ли шум (true/false)

+ файл run.m генерирует изображение и выводит его на экран
